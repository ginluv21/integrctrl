# integrctrl

Утилита командной строки на языке Си (C99) для Linux. Проверяет целостность файлов через MD5-хеши.

Курсовая работа по дисциплине «Программирование», СибГУТИ, вариант 2.5.

## Запуск

```
integrctrl -s -f <база> <директория>      # сохранить хеши
integrctrl -s -r -f <база> <директория>   # то же, рекурсивно
integrctrl -c -f <база> <директория>      # проверить целостность
```

Результат проверки для каждого файла: `OK`, `CHANGED`, `MISSING` или `NEW`.

## Сборка

```
make
```

Исполняемый файл появится в `build/integrctrl`.

## Структура проекта

```
src/
  integrctrl.c   - основной код программы
  md5.c          - реализация алгоритма MD5
include/
  integrctrl.h   - структуры данных
  md5.h          - заголовок MD5
Makefile
```
